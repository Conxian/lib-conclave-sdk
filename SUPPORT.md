# Support for lib-conclave-sdk

Thank you for using the Conclave SDK. We are committed to providing a secure and high-performance infrastructure for hardware-backed key management.

## Support Channels

### 1. GitHub Issues (Public)
For technical questions, bug reports, and feature requests, please use [GitHub Issues](https://github.com/Conxian/lib-conclave-sdk/issues).
- Use the provided templates for Bug Reports and Feature Requests.
- Search existing issues before opening a new one.

### 2. Linear (Partners)
If you are a partner or have been granted access to the Conxian-Labs Linear workspace, please track high-priority items there. This is the fastest way to reach the core engineering team for non-security urgent issues.

### 3. Security Reports (Private)
**DO NOT use public issues for security vulnerabilities.**
Please report all security-related issues to:
- **Email**: security@conxian.com
- **Linear**: Use the "Security" label if you have workspace access.

## Service Level

- **Current Status**: Beta (`0.x`).
- **Response Time**: We aim to respond to critical issues within 2 business days. General inquiries are handled on a best-effort basis.
- **Breaking Changes**: During the `0.x` phase, breaking changes may occur in minor releases. Please follow the [Changelog](CHANGELOG.md) for migration guidance.
